1. Requirements
    2차원 평면의 점의 위치를 랜덤으로 n 개 생성했을때, 
    두 점간의 거리가 10 이내인 개수 찾아 출력하는 c++ 프로그램을 작성하시오.

2. Condition
두개의 클래스 구성
    1. 점의 위치를 나타내는 클래스
    2. 점들의 집합을 나타내는 클래스

3. detail design
	1. 점의 위치를 나타내는 클래스
		데이터 : 점의 위치 (x,y)
		메소드 : 점의 위치 중복가능
				거리 계산
				x위치가져오기
				y위치가져오기
				plotXY(x,y)

	2. 점들의 집합을 나타내는 클래스
		데이터 : 점의 위치 array(x,y)
		매소드 : database에 추가
				점들간 거리
				결과 출력
				plotAll()

4. main() for testing

	점들간 거리 계산
	결과출력
